HTML/CSS Cheatsheet

HTML Elements

HTML elements are a type of HTML document component. They are composed of opening and closing tags, as well as in between content.

Element Name Starting Tag Content Ending Tag Function
doctype <!DOCTYPE > - - Contains information on page document type
html <html> All HTML document elements </html> Contains all HTML document content
head <head> document info elements </head> Contains document info/metadata
body <body> Main page content </body> Contains main written/visual information
division <div> section elements </div> Creates document section
h1-h6 <h1> header text </h1> Text headers
paragraph <p> paragraph text </p> Encloses paragraph text
emphasis <em> italicized text </em> Italicize
strong <strong> bold text </strong> Bold
line break <br> - - Adds a line break
unordered list <ul> list item elements </ul> Creates bullet list
ordered list <ol> list item elements </ol> Created numbered list
list item <li> item text </li> List item in list
anchor <a> link text </a> Creates link
span <span> text </span> Inline text sectioning
image <img /> - - Adds an image
video <video> Unsupported error text </video> Adds a video

HTML Attributes

HTML attributes are located in an HTML elements opening tag. They add information to and modify the behavior of the elements.

Tag Attributes Function
any class="" Add element to the class
any id="" Adds id to an element
<a>, <link> href="" Link to url, other element (#element)
<img>, <video> src="" Link to url
<img> alt="" alternative text describing image
<a> target="_blank" Opens link in new tab
<video> controls Adds control buttons to video
